# MindWell AI 🧠✨

An innovative AI-powered mental health support platform for young adults, providing 24/7 personalized wellness guidance, mood tracking, and evidence-based therapeutic techniques.

## Features 🎯

- **AI Chat Support** - Talk to an AI that listens and understands your emotional needs
- **Mood Tracking** - Daily mood check-ins with personalized feedback and visual insights
- **Guided Meditations** - Calming exercises tailored to your stress levels
- **Journal & Notes** - Reflective journaling to track your emotional journey
- **Task Management** - Organize your wellness goals and daily tasks
- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **24/7 Availability** - Access support whenever you need it

## Tech Stack 🛠️

- **Frontend**: HTML5, CSS3, Bootstrap 5
- **Scripting**: Vanilla JavaScript

## Getting Started 🚀

### Prerequisites

- Modern web browser (Chrome, Firefox, Safari, Edge)
- No server-side setup required (fully client-side)

### Installation

1. **Clone the repository:**

```bash
git clone https://github.com/your-username/mindwell-ai.git
cd mindwell-ai
```

2. **Open in browser:**
   - Simply open `index.html` in your web browser
   - Or use a local server:
   ```bash
   python -m http.server 8000
   # Then visit http://localhost:8000
   ```

## Features Breakdown 🎨

### Responsive Design

- Desktop (1024px+): Full layout with 40×40px logo
- Tablet (768px–1023px): Optimized for iPad
- Mobile (<768px): 28×28px logo with touch-friendly UI
- Small phones (<480px): 24×24px logo, compact layout

### Mood Tracker

- 10 personalized wellness questions
- 5-point scale ratings (1=Very Low, 5=Excellent)
- Automatic feedback messages based on average score
- Line chart visualization of mood trends over time
- Data persists across sessions using localStorage

### Journal Section

- Write and save journal entries
- View full entry text with timestamps
- Delete entries you no longer need
- Newest entries displayed first
- Centered, easy-to-read layout

### Task Management

- Add new tasks to your to-do list
- Mark tasks as completed
- Move completed tasks to a separate section
- Delete tasks
- Persistent storage with user profile
